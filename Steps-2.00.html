How to set up the dev environment?

Please follow the following steps to install Power BI Desktop on your computer.

Installing Power BI Desktop
There are two ways to start the download process:
1. Go to the Power BI Desktop webpage and click the download button

2. Click the download button located under the down-arrow tab in the online Office 365 version of Power BI. Either way will download an .msi file to your PC.

3. You can also get Power BI Desktop from the Microsoft Store

4. Run that file once the download is complete. After accepting the license agreement, you will be able to choose where you want to install the application. In most cases, the default location will be adequate. The installation process will take about a minute, depending on your PC and network connection.

5. When you start the Power BI Desktop application the first time, it will go through an initialization process that takes a little longer than you might expect.

6. Now go ahead and create a report that is going to be published later on.
